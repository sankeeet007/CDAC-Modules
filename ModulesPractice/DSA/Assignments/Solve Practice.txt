1. Find the second largest element in an array without sorting it.
2. Rotate an array to the right by k positions in-place.
3. Move all zeros to the end of an array while preserving relative order of non-zero elements.
4. Find the duplicate number in an array of n+1 integers where values are in range [1, n].
5. Rearrange an array such that positive and negative numbers alternate, starting with a positive number.
6. Given an array of integers, find the smallest positive integer missing from it. Array may contain negatives and duplicates.
7. Find the majority element in an array — the element that appears more than n/2 times. Assume it always exists.
8. Find the equilibrium index of an array — an index where the sum of elements to its left equals the sum to its right.
9. Given an unsorted array, find the minimum number of swaps required to sort it.
10. Given an array where every element appears twice except one, find the element that appears only once.
