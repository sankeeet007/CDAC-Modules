Word Problem: Snacks Center Menu Management System (Sorted Linked List)
A snacks center wants to develop a menu management system to efficiently manage its food items. The system should be implemented using a linked list, so that items can be dynamically added, deleted, and updated.
	Each menu item contains:
	Item ID
	Item Name
	Price
Snack Center Menu Management System (using Linked List)
The system should provide a menu-driven interface with the following operations:
Add Menu Item
	Insert a new snack item into the menu.
	The user should enter item ID, name, and price.
	The item must be inserted in such a way that the list remains sorted (sorted Item Name).
Delete Menu Item
	Remove a snack item from the menu using its Item ID.
	Display a message if the item is not found.
Display Menu
	Display all snack items in the menu in sorted order.
Modify Menu Item
	Update the details (name and/or price) of an existing item using its Item ID.
	After modification, ensure the list remains sorted.
	Display a message if the item does not exist.
